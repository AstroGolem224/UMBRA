# UMBRA — Unified Management Board for Runtimes & Agents

**Schatten-Interface für CMG's Agent-Ökosystem.**  
Das Zentrum von dem aus Druid alle Agents, Tools, Projekte und Skills steuert.

## 🌒 Vision

UMBRA ist die unsichtbare Schaltzentrale hinter PRISM, Forge und allen anderen CMG-Agents.  
Ein einheitliches Interface das:

- **Agent Registry** — Alle verfügbaren Agents + Status + Skills
- **Tool Access** — Welche Tools welcher Agent nutzen darf
- **Project Dashboard** — Status aus PM Tool, GitHub, GAP  
- **Skills Browser** — Verfügbare Skills, Prompt-Varianten, Settings
- **IDE Launcher** — Direkter Start von VSCode, Godot, etc.
- **Extensions** — Plugins für jedes externe System

1. **Agent Overview** — Liste aller Agents + Live-Status
